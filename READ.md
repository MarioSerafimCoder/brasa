# BRasa

Uma plataforma pessoal para gerenciamento e reprodução da minha biblioteca de filmes.

## Objetivos

- Interface inspirada em Apple TV e Plex
- Biblioteca pessoal em 4K
- Integração futura com Jellyfin
- Player próprio
- Busca inteligente
- Coleções
- Favoritos
- Continuação automática

## Stack

- HTML5
- CSS3
- JavaScript ES Modules
- Lucide Icons
